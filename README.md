# Orioles Summer 2026 - Stat Tracker

Valley Wheel Over-28 Baseball League stat tracker app.

## How to Deploy (Step by Step)

### 1. Create a GitHub Account
- Go to https://github.com and sign up (free)
- Verify your email

### 2. Create a New Repository
- Click the green "New" button (or go to https://github.com/new)
- Name it: `orioles-tracker`
- Keep it Public
- Do NOT check "Add a README" (we already have one)
- Click "Create repository"

### 3. Upload the Project Files
- On your new empty repo page, click "uploading an existing file"
- Drag the ENTIRE contents of this folder into the upload area:
  - `src/` folder (contains main.jsx and App.jsx)
  - `public/` folder (contains manifest.json, sw.js, icons)
  - `index.html`
  - `package.json`
  - `vite.config.js`
  - `vercel.json`
  - This `README.md`
- Click "Commit changes"

### 4. Deploy on Vercel
- Go to https://vercel.com and click "Sign Up"
- Sign up with your GitHub account
- Click "Add New Project"
- Import your `orioles-tracker` repository
- Framework Preset: select "Vite"
- Click "Deploy"
- Wait about 60 seconds

### 5. Your App is Live!
- Vercel gives you a URL like: `orioles-tracker.vercel.app`
- Text this link to your team
- Everyone opens it and taps "Add to Home Screen"

## Adding to Home Screen

**iPhone:** Open link in Safari > tap Share button > "Add to Home Screen"

**Android:** Open link in Chrome > tap menu (three dots) > "Add to Home Screen"

The app will look and feel like a native app with the Orioles icon.
